# Changelog

All notable changes to SkillPilot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Modern dashboard UI with statistical overview cards
- Parallel career path creation and management
- Persistent skill tracking across career paths
- Google OAuth authentication via NextAuth.js
- Dark mode support with theme toggle
- Responsive design for mobile and tablet devices
- Glass-morphism UI components
- Career path comparison features
- Real-time progress tracking with visual charts
- Framer Motion animations throughout the app

### Changed
- Migrated to Next.js 15 with App Router
- Updated to React 19
- Modernized color scheme with blue, green, and pink accents
- Improved navigation with sticky navbar
- Enhanced card designs with soft shadows

### Fixed
- Authentication flow improvements
- Database connection optimization
- Type safety across all components

## [0.1.0] - 2026-02-07

### Added
- Initial project setup with Turborepo monorepo
- Next.js application structure
- Prisma database integration
- Basic authentication system
- Core career planning features
- Project documentation
